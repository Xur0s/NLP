Usage is to train different language models and evaluate them on a test corpus. Where pre-processing is done to
the corpus and then trained to answer questions.

Running the file will output two files called "1.2Answers.txt" and "1.3Answers.txt".
1.2 answers file will have all of the dictionaries of all of the models. 
1.3 answers file has all of the questions from 1.3 and their corresponding answers.
To get these files to output, simply run the file, then check the results.
